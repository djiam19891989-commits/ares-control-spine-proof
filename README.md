# ARES Control Spine Proof Pack

ARES is a proprietary governed autonomous engineering runtime built by Dylan Campbell in Queensland, Australia.

This public repository contains a proof/demo package showing the ARES governance spine:

- Validate decides
- MRVL gates
- Operator approvals/rejections are recorded
- Execution risk policy classifies high-risk actions
- Evidence receipts prove governance decisions
- One-command verification runner demonstrates the proof stack

This repository is **not** the full ARES source code.

The ARES runtime, orchestration engine, provider routing, execution layer, approval enforcement internals, and commercial licensing framework remain proprietary and are not included in this public proof package.
