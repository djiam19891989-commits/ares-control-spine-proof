# ARES Control Spine Proof Pack v0.1

## Release

`v0.1-public-proof-pack`

## Purpose

This is the first public proof package for the ARES governance/control spine.

It demonstrates the documented control model:

Validate decides → MRVL gates → Operator approves → Execution policy classifies risk → Evidence receipts prove the path.
