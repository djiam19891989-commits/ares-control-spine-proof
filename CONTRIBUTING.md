## Contribution Policy
- Contributions are limited to individuals who have been granted access to the repository. Please contact the repository owner to request access.
- All contributions must adhere to the coding standards and practices defined by the project maintainers.
- Before submitting a pull request, please ensure that your code has been tested and that it adheres to the project's style guidelines.
- Any contributions must be made via pull requests with a clear description of the changes proposed.
- Contributors must not submit any proprietary or confidential information in their contributions.

Thank you for your interest in contributing to the Ares Control Spine Proof project!
